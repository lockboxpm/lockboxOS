# Push Git Workflow

1. **Stage all changes**
   ```bash
   git add -A
   ```
2. **Commit** (replace the commit message as needed)
   ```bash
   git commit -m "Your commit message"
   ```
3. **Push to remote** (adjust branch if needed)
   ```bash
   git push origin main
   ```
